## üé® Design Guidelines

### **Color Palette to Match**:
- Primary: `#667eea` (Blue)
- Secondary: `#764ba2` (Purple)
- Background: `#1e1b4b` to `#312e81` (Dark gradient)
- Text: `#1a202c` (Dark) / `#64748b` (Medium)

### **Style Consistency**:
- **Modern, clean interface design**
- **Glass-morphism effects** (blur, transparency)
- **Gradient accents** matching brand colors
- **High contrast** for accessibility
- **Professional but approachable** aesthetic

### **Technical Specifications**:
- **Format**: JPG for photos, PNG for UI screenshots with transparency
- **Compression**: 80-85% quality for web optimization
- **Color Space**: sRGB for web display
- **Resolution**: 72 DPI minimum for web

## üì± Responsive Considerations

### **Mobile Optimization**:
All images should work well when scaled down to mobile sizes:
- **Hero**: Will be cropped to center focus
- **Features**: Will stack vertically on mobile
- **Use Cases**: Will display in single column

### **Retina Display Support**:
Consider providing 2x versions for high-DPI displays:
- `hero-background@2x.jpg` (4800x3200px)
- `location-discovery@2x.jpg` (2400x1600px)
